Mock data for Front-end development

By Thien Nguyen <nguyenphongthien@gmail.com>

  • Step 1: Include mock.js to file script

  • Step 2: Add mock data:

    • Simply add ~datatype~ anywhere we want to use mock data, e.g:
    • <div>~s:10,50~</div>
    • NOTE: Every mock data have to be wrapped in a "mock" class, if you want to use this in all the page, just simply add class "mock" to the body tag
    • (eg: <span class="mock">~s:40,100~</span>)

  • Mock data usage:

    All the examples below will be used this plugin as when you refresh the page, you'll see the different result. Try it!

    • Name

      • name: ~nm~
        (eg: "~nm~")
      • male name: ~nm:m~
        (eg: "~nm:m~")
      • female name: ~nm:f~
        (eg: "~nm:f~")
      • firstname: ~fnm~
        (eg: "~fnm~")
      • male firstname: ~fnm:m~
        (eg: "~fnm:m~")
      • female firstname: ~fnm:f~
        (eg: "~fnm:f~")
      • lastname: ~lnm~
        (eg: "~lnm~")

    • String

      • string with exact length: ~s:length~
        (eg: ~s:100~ => "~s:100~")
      • string with random length: ~s:min_length,max_length~
        (eg: ~s:100,300~ => "~s:100,300~")

    • Image

      • image with exact width, exact height: ~i:width|height~
        (eg: ~i:200|200~ => ~i:200|200~)
      • image with exact width, random height: ~i:width|min_height,max_height~
        (eg: ~i:200|100,150~ => ~i:200|100,150~)
      • image with random width, exact height: ~i:min_width,max_width|height~
        (eg: ~i:100,300|100~ => ~i:100,300|100~)
      • image with random width, random height: ~i:min_width,max_width|min_height,max_height~
        (eg: ~i:200,300|100,150~ => ~i:200,300|100,150~)
      • image with topic: ~i:min_width[,max_width]|min_height[,max_height]|keyword~
        (eg: ~i:200,300|100,150|people~ => ~i:200,300|100,150|people~)

        Keyword can be:

        • abstract
        • animals
        • business
        • cats
        • city
        • food
        • night
        • life
        • fashion
        • people
        • nature
        • sports
        • technics
        • transport

    • Number

      • interger number: ~nb:min,max~
        (eg: ~nb:100,1000~ => ~nb:100,1000~)
      • float number: ~nb:min,max|number_after_dot~
        (eg: ~nb:100,1000|4~ => ~nb:100,1000|4~)
